MotorCtrlクラスの取扱説明書
======================================

    初期設定
    ----------------------------------
    1. 上部のdefineに使用するピン番号を書く

    2. クラスをグローバルで宣言する　
        例)　MotorCtrl motor // 名前は好きにしてどうぞ


    各オブジェクトの使い方
    ----------------------------------

    < setOperationThreshold(int percent) について >
    これはコントローラのしきい値を決めるやつ. 仮に30と入れると根本から3割が判定から除外される.
    初期値は20なのでいじりたければいじること

    使用例) className.setOperationThreshold(20); // 0~100で値を指定


     < move(const unsigned char x,const unsigned char y); について >
    引数はアナログスティックのx, y の値(0-255) を指定する.
    内部でベクトルの合成を行ってくれる.

    使用例) className.move(LX, LY); // 内部でｘ,ｙは -128され値を補正される


     < void stop() の使い方 >
     動きを止めてくれる. 非常停止用

     使用例) className.stop() // 止まる
